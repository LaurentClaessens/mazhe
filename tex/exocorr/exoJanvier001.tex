\begin{exercice}\label{exoJanvier001}

Esquisser dans le système d'axes le graphe de la fonction indiquée :
\begin{enumerate}

\item
$x^2+1$

\item
$x^3$

\item
$\tan(x)$

\item
$\log_2(x)$

\item
$| \ln(x) |$

\item
$\ln| x |$

\item
$\sin(x+1)$

\item
$\sin(2\pi x)$

\item
$ e^{-x}$.

\end{enumerate}
