	La fonction
	\begin{equation}
		f(x)= e^{-x^2}\int_0^x e^{t^2}dt
	\end{equation}
	est appelée \emph{intégrale de Dawson}.

	\begin{enumerate}

		\item
			Calculer $f(1)$
		\item
			Déterminer une solution de l'équation $f(x)=0.5$. (Indication : cette solution est proche de $0.7$)

	\end{enumerate}
