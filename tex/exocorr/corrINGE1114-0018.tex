\begin{corrige}{SerieUn0018}

	Pour $n=1$, on a $1+x\geq 1+n$, ce qui est manifestement vrai. Pour $n+1$, nous avons
	\begin{equation}
		\begin{aligned}[]
			(1+x)^{n+1}=(1+x)(1+x)^n&\geq (1+x)(1+nx)\\
							&=1+nx+x+\underbrace{nx^2}_{\geq 0}\\
							&\geq 1+x(n+1),
		\end{aligned}
	\end{equation}
	ce qui est bien la thèse avec $n+1$.

\end{corrige}
