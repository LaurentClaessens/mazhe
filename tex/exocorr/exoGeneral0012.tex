\begin{exercice}\label{exoGeneral0012}

Déterminer les extréma des fonctions suivantes :
\begin{enumerate}

\item
$y=2\sin(x)+\cos(2x)$

\item
$y=x^2+250/x$

\item
$y=e^x/x$.

\end{enumerate}

\end{exercice}
