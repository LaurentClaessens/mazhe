\begin{exercice}\label{exoInter0012}

Exercice 23, page 51. Calculer la longueur des arcs de courbe suivants.
\begin{enumerate}

\item
$y=\ln(1-x^2)$ pour $0\leq x\leq \frac{ 1 }{2}$
\item
$y=x^{3/2}$ pour $0\leq x\leq 5$
\item
$y=\ln(x)$ pour $1\leq x\leq 2\sqrt{2}$.

\end{enumerate}
