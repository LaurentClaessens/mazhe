\begin{exercice}\label{exoEqsDiff0006}

Trouvez toutes les solutions des équations différentielles suivantes et des problèmes de Cauchy lorsqu'ils sont spécifiés.
\begin{multicols}{2}
\begin{enumerate}
\item
$y'' - 2y' = 0$\\
$y(0) = 0, y'(0)=1$
% (3t + 1)(t+1) = 0
\item 
$  3y'' +4y' +y = 0 $
\item
 $ y'' - 6y' + 9y = 0$ \\
$y(1) = 0, y'(1)=1$
\item 
$  y'' - 4y' + 4y = 0  $
\item 
$y'' + y = 0$ \\
$y(3) = 0, y'(3)=1 $
\item
 $  y'' -2y' + 3y = 0 $
\item
 $  y^{4/} - 2y''' - 3y'' + 8y' - 4y = 0 $
\end{enumerate}
\end{multicols}

\end{exercice}
