\begin{exercice}\label{exoLineraire0002}

Exercice 7, page 86. Discuter les systèmes suivants
\begin{enumerate}

\item
$\left\{
\begin{array}{ll}
mx+y=1\\
x+my=1
\end{array}
\right.$

\item
$\left\{
\begin{array}{ll}
\lambda x+y+m=1\\
x+\lambda y+z=1\\
x+y+\lambda z=1
\end{array}
\right.$

\end{enumerate}
