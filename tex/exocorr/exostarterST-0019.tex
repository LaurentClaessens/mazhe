\begin{exercice}\label{exostarterST-0019}

  Vérifier que $y(x) = -\sqrt{x^2+1}$ est l'unique solution sur $\mathbb{R}$ de l'équation 
    \begin{equation}
      2yy'-2x = 0,
    \end{equation}
    qui satisfait la condition initiale $y(0)=-1$.

\end{exercice}
