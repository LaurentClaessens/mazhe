\begin{exercice}\label{exoEquaDiff0001}

Résoudre les équations suivantes pour $y(x)$.
\begin{enumerate}

\item
$y'\sqrt{1-x^2}+xy=0$,
\item
$(1-y^2)dy-ydx=0$.

\end{enumerate}
