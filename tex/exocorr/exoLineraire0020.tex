\begin{exercice}\label{exoLineraire0020}

	Exercice 9, page 81. Quelles sont les composantes du vecteur $(1,2)\in\eR^2$ dans les bases suivantes ?
	\begin{enumerate}

		\item
			$\{ (1,0),(0,1) \}$,
		\item
			$\{ (0,1),(1,0) \}$,
		\item
			$\{ (1,2),(1,1) \}$,
		\item
			$\{ (1,1),(2,1) \}$,
		\item
			$\{ (-1,3),(4,0) \}$

	\end{enumerate}
