\begin{exercice}\label{exo_II-1-05}

Résoudre les équations différentielles suivantes :
\begin{enumerate}
\item $ \sin(2t)y'+y^2=1$,
\item $y'+\frac{ y }{ t+1 }=\sin(t)$,
\item $y'-\frac{ a }{ t }y=e^tt^a$, où $a$ est une constante,
\item $yy'+(1+y^2)\sin(t)=0$,
\item $y''+2y'+y= e^{-t}\ln| t |$.
\end{enumerate}
